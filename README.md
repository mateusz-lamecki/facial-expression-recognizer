# facial-expression-recognizer

### Dataset
For training the model dataset from [Facial Expression Recognition Challenge](https://www.kaggle.com/c/challenges-in-representation-learning-facial-expression-recognition-challenge) was used.

### Installation
Download fer2013.tar.gz file from FERC competition and unpack it in data/ directory.

### Training
To train the model run Train.ipynb script. Training takes ~2 minutes on computer with GPU support. The model will be saved in models/ directory.

### Face recognition
Run ``` camera.py ```
